FindEcoEvoCycle

FindEcoEvoCycle[traits, pops]
finds an evolutionary equilibrium with an ecological cycle, with initial guess traits and pops.

DetailsDetails

  • FindEcoEvoCycle currently only works on continuous-time periodically forced models. Eventually it will also handle unforced models with eco-evolutionary limit cycles.
  • The following options can be given:
  • Method"FindRoot"method to find result
    FindRootOpts{}options to pass to FindRoot
    NDInvOpts{}options to pass to NDInv
    EcoSimOpts{}options to pass to EcoSim
    EcoEvoSimOpts{}options to pass to EcoEvoSim
    FitnessGradientAutomatichow to calculate the fitness gradient
    Fixed{}fixed variables
    TraitShiftRate{}rate of environmental change
    MaxIterations100max number of iterations
    AccuracyGoal6accuracy goal
    MonitorFalseevaluation monitor
    PrintTraceFalseprint a trace of steps
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose to all contained functions
  • Possible settings for the Method option are:
  • "FindRoot"use FindRoot
    "FixedPoint"use FixedPoint
  • Possible settings for the FitnessGradient option are:
  • Automatic"EcoEvoSim" if unstructured guilds, "NDInv" otherwise
    "NDInv"use NDInv
    "EcoEvoSim"use EcoEvoSim
  • $FindEcoEvoCycleSteps counts the number of steps taken.
  • traits and pops can be combined.
