SelectEcoStable

SelectEcoStable[sol]
selects stable equilibria from sol.

SelectEcoStable[traits, sol]
uses trait values traits.

DetailsDetails

  • The following options can be given:
  • VerboseFalseprint out extra info
    Method"Automatic"method to test stability
    SimplifyTruesimplify the results
    Timetfix time if nonautonomous model
    EcoEigenvaluesOpts{}options to pass to EcoEigenvalues
    Assumptions{}assumptions to make in simplifying
  • Possible settings for the Method option are:
  • "Automatic"if ContinuousTime and three or fewer equations "RouthHurwitz", otherwise "Eigenvalues"
    "Eigenvalues"use EcoEigenvalues
    "RouthHurwitz"use RouthHurwitzCriteria
  • Possible settings for the Simplify option are:
  • Trueapply Simplify to results
    Realapply RealSimplify to results
    Fullapply FullSimplify to results
    Falsedon't simplify results
