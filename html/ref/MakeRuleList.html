MakeRuleList

MakeRuleList[var, n, val]
makes a list of n rules where var takes on value val.

MakeRuleList[var, ns, val]
makes a list of rules with dimensions ns where var takes on value val.

MakeRuleList[var, n, {min, max}]
makes a list of n rules where var ranges from min to max.

MakeRuleList[vars, n, vals]
makes a list of rules between corresponding elements of the lists vars and vals.

MakeRuleList[{var1, …, varnv}, {n1, … , nnv}, {{min1, max1}, … , {minnv, maxnv}}]
makes a list of rules with dimensions ns where {class -> TI} ;; var ;; XMLElement[sub, {}, {i}] ranges from mini to maxi.
