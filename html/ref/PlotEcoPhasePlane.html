PlotEcoPhasePlane

PlotEcoPhasePlane[{var1, var1min, var1max}, {var2, var2min, var2max}]
plots ecological streams and isoclines in the phase plane of variables var1 and var2.

PlotEcoPhasePlane[traits, {var1, var1min, var1max}, {var2, var2min, var2max}]
uses trait values traits.

DetailsDetails

  • PlotEcoPhasePlane combines PlotEcoIsoclines and PlotEcoStreams to make a phase plane diagram.
  • The following options can be given:
  • Fixed{}fixed variables
    Timetfix time if nonautonomous model
    FrameLabelAutomaticlabels for x- and y-axes
    PlotEcoIsoclineOpts{}options to pass to PlotEcoIsoclines
    PlotEcoStreamsOpts{}options to pass to PlotEcoStreams
    VerboseFalseprint out extra info
    VerboseAllFalseapply Verbose recursively
  • In addition, any options for PlotEcoIsoclines and PlotEcoStreams can be used.
  • FrameLabel is automatically generated by default (turn off with FrameLabelNone).
