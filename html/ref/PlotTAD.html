PlotTAD

PlotTAD[traits, pops]
plots abundance vs trait for the species in traits and pops.

PlotTAD[traitsandpops]
uses combined community state traitsandpops.

DetailsDetails

  • The following options can be given:
  • LoggedFalsewhether to log the y-axis
    GuildAutomaticwhich guild to plot
    TraitAutomaticwhich trait to plot
    ComponentAutomaticwhich component to plot
    MinPop0minimum population size
    MarkerStyleAutomatichow to style species markers
