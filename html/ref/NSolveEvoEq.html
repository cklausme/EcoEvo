NSolveEvoEq

NSolveEvoEq[sol]
numerically solves for evolutionary equilibria using ecological equilibrium sol.

DetailsDetails

  • The following options can be given:
  • DelayDInvFalseevaluate NDInv numerically at each point
    DInvOpts{}options to pass to DInv / NDInv
    NSolveOpts{}options to pass to NSolve
    EvoEquation"QG"equation for trait dynamics
    BoundaryDetectionFalseforce traits in traitrange
    TraitShiftRate{}rate of environmental change
    Fised{}fixed variables (not written yet)
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for the EvoEquation option are:
  • "QG"quantitative genetics (breeders equation)
    "CE"canonical equation of adaptive dynamics
