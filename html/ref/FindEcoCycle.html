FindEcoCycle

FindEcoCycle[pops]
finds an ecological limit cycle using initial guess pops.

FindEcoCycle[traits, pops]
uses trait values traits.

DetailsDetails

  • The following options can be given:
  • MethodAutomaticmethod to find cycle
    Fixed{}fixed variables
    FindRootOpts{}options to pass to FindRoot
    EcoSimOpts{}options to pass to EcoSim
    MaxIterations100maximum number of iterations to make
    AccuracyGoals6AccuracyGoal for stopping search
    LoggedFalsewhether to log transform Extensive variables
    WarmUp1000time in initial warmup (method "EcoSim")
    WarmUp2100time to find maxima (method "EcoSim")
    WarmUp30.1time to advance beyond last maximum (method "EcoSim")
    TMax1000post-warmup time to look for cycle (method "EcoSim")
    TriggerVariableAutomaticvariable to look for cycle with (method "EcoSim"); automatic = first variable
    WhenEventOpts{"DetectionMethod""Interpolation","LocationMethod"{"Brent",MaxIterations1000}}}options to pass to WhenEvent (method "EcoSim")
    PeriodAutomaticinitial guess to period (method "FindRoot", unforced ContinuousTime model); fixed period (unforced DiscreteTime model)
    MonitorFalsepopup progress monitor window
    PrintTraceFalseprint variables at each step
    VerboseFalseprint out extra info
    VerboseAllFalseapply Verbose recursively
  • Variables not included in the initial guess are fixed at zero.
  • Possible settings for the Method option are:
  • Automatic"EcoSim" for unforced ContinuousTime models, "FixedPoint" otherwise
    "EcoSim"for unforced ContinuousTime models
    "FixedPoint"for DiscreteTime or forced ContinuousTime models
    "FindRoot"for all model types
  • Method "EcoSim" has four stages: 1) initial warmup to get on limit cycle, 2) collecting maxima of the TriggerVariable, 3) a slight advance past max last maximum, 4) solution for up to TMax that uses WhenEvent to find return to initial value and tests that entire system is within 10^-AccuracyGoal of previous maximum.
  • Method "FixedPoint" repeatedly solves the model for a Period until final value is within 10^-AccuracyGoal of initial value.
  • Method "FindRoot" applies FindRoot to the difference between the initial and final values over one Period. In the case of unforced ContinuousTime models, Period is also determined numerically.
  • Each call to EcoSim increments $findecocyclesteps.
  • LoggedTrue automatically sets EcoSim option LoggedTrue, unless overridden by EcoSimOpts.
