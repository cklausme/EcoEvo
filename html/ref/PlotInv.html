PlotInv

PlotInv[traits, sol, {trait, traitmin, traitmax}]
plots a fitness landscape, where trait varies from traitmin to traitmax.

DetailsDetails

  • The following options can be given:
  • InvOpts{}options to pass to Inv
    Fixed{}fixed variables
    DelayInvAutomaticre-evaluate Inv numerically at each point
    ShowSpecies"TAD"how to indicate species
    PlotTADOpts{}options to pass to PlotTAD
    TADVerticalScale0.25proportion of vertical extent to scale TAD
    MarkerStyleAutomaticstyle of species markers (ShowSpecies"Axis")
    AxesLabelAutomatichow to label axes
    Timetfix time if nonautonomous model
    PlotStyleGrayplot style
    PlotPoints5initial number of plot points
    MonitorFalsepopup progress monitor window
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for the ShowSpecies option are:
  • "TAD"plot a TAD indicating species abundance
    "Axis"put points on the axis indicating species
    Falsedon't show species
  • The guild of the invader is inferred from trait.
