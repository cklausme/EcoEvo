PlotZIP

PlotZIP[{var1, var1min, var1max}, {var2, var2min, var2max}, inv]
creates a zero invasion plot, with invader inv (inferred if omitted).

PlotZIP[eq0, {var1, var1min, var1max}, {var2, var2min, var2max}, inv]
uses equilibrium eq0.

DetailsDetails

  • The following options can be given:
  • FindEcoAttractorOpts{}options to pass to FindEcoAttractor
    InvOpts{}options to pass to Inv
    DelayInvFalsere-evaluate Inv numerically at each point
    PlotType"ZIP"type of plot
    ICs{}initial conditions to pass to FindEcoAttractor
    Fixed{}fixed variables
    InvThreshold0threshold for successful invasion
    PlotOpts{MaxRecursion  3}
    BoundaryStyleBlackstyle of invasion boundary
    InvStyleGraycolor to indicate positive growth
    NonInvStyleWhitecolor to indicate negative growth
    MonitorFalsepopup progress monitor window
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for the PlotType options are:
  • "ZIP"zero invasion plot made using ContourPlot
    "RegionZIP"zero invasion plot made using RegionPlot
    "ContourPlot"plot all invasion contours
    "Plot3D"3D plot of invasion fitness
