PlotPIP

PlotPIP[{trait, traitmin, traitmax}]
creates a pairwise invasibility plot, where trait varies from traitmin to traitmax.

PlotPIP[sol, {trait, traitmin, traitmax}]
uses equilibrium sol for improved speed.

DetailsDetails

  • The following options can be given:
  • PlotType"PIP"type of plot
    FindEcoAttractorOpts{}options to pass to FindEcoAttractor
    ICs{}initial conditions to pass to FindEcoAttractor
    InvOpts{}options to pass to Inv
    InvThreshold0threshold for successful invasion
    DelayInvFalsere-evaluate Inv numerically at each point
    PlotOpts{MaxRecursion3}options to pass to ContourPlot / RegionPlot
    FrameLabelAutomatichow to label axes
    InvStyleGraycolor to indicate successful invasion
    NonInvStyleWhitecolor to indicate unsuccessful invasion
    BoundaryStyleBlackstyle of invasion boundary
    ZeroDiagonalFalseforce diagonal to equal zero
    SubtractDiagonalFalsesubtract diagonal from all points
    Fixed{}fixed variables
    MonitorFalsepopup progress monitor window
    VerboseFalseprint extra info
    VerboseAllFalseApply Verbose recursively
  • Possible settings for the PlotType option are:
  • "PIP"PIP made using ContourPlot
    "RegionPlot"PIP made using RegionPlot
    "Plot3D"3D plot
    "ContourPlot"contour plot of invasion fitness
