PrestonPlot

PrestonPlot[pops]
makes a Preston species abundance distribution plot based on pops.

DetailsDetails

  • The following options can be given:
  • GuildAutomaticwhich guild to plot
    ComponentAutomaticwhich component to plot
    Base10base for log transformation
    MinPop0minimum population size
    Bandwidth"Scott"bandwidth specification to pass to SmoothKernelDistribution
    ShowSpeciesTruewhether to put species markers on x-axis
    MarkerStyleAutomatichow to style species markers
    PlotRange{0,All}plot range
  • In addition, any valid Plot options are passed to Plot and any valid ListPlot options are passed to ListPlot.
