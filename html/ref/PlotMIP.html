PlotMIP

PlotMIP[{trait, traitmin, traitmax}]
creates a mutual invasibility plot, where trait varies from traitmin to traitmax.

PlotMIP[sol, {trait, traitmin, traitmax}]
uses equilibrium sol for improved speed.

PlotMIP[{trait1, trait1min, trait1max}, {trait2, trait2min, trait2max}]
allows for two different traits.

PlotMIP[{sol1, sol2}, {trait1, trait1min, trait1max}, {trait2, trait2min, trait2max}]
uses equilibria sol1 and sol2.

DetailsDetails

  • The following options can be given:
  • PlotType"MIP"type of plot
    FindEcoAttractorOpts{}options to pass to FindEcoAttractor
    ICs{}initial conditions to pass to FindEcoAttractor
    InvOpts{}options to pass to Inv
    InvThreshold0threshold for successful invasion
    DelayInvFalsere-evaluate Inv numerically at each point
    PlotOpts{MaxRecursion3}options to pass to ContourPlot / RegionPlot
    FrameLabelAutomatichow to label axes
    InvStyleGraycolor to indicate successful invasion (MIP)
    NonInvStyleWhitecolor to indicate unsuccessful invasion
    BoundaryStyleBlackstyle of invasion boundary
    SpeciesColors{Red,Blue}colors for species 1 & 2 (outcome plots)
    ZeroDiagonalFalseforce diagonal equal to zero
    Fixed{}fixed variables
    MonitorFalsepopup progress monitor window
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • Possible setting for the PlotType option are:
  • "MIP"MIP made using ContourPlot
    "RegionMIP"MIP made using RegionPlot
    "Outcome"competition outcome plot made using ContourPlot
    "RegionOutcome"competition outcome plot made using RegionPlot
