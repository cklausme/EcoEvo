EcoSim

EcoSim[pops, tmax]
simulates ecological dynamics, with initial densities in pops, from time t=0 to tmax.

EcoSim[traits, pops, tmax]
uses trait values traits.

DetailsDetails

  • EcoSim uses NDSolve for ContinuousTime models and RecurrenceTable for DiscreteTime models.
  • The following options can be given:
  • NDSolveOpts{}options to pass to NDSolve
    LoggedFalselog transform the model before solution
    InterpolationOrder7InterpolationOrder when untransforming logged model
    InterpolationPoints1000InterpolationPoints when untransforming logged model
    Fixed{}fixed variables
    WhenEvents{}WhenEvents to pass to NDSolve
    TimeScale1change of timescale
    Output"Dynamics"output type
    OutputTMin0when to start results
    VerboseFalseprint out extra info
  • Possible settings for the Output option are:
  • "Dynamics"temporal rule lists
    "FinalSlice"final values
  • Omitting a variable from the initial conditions is the same as fixing it at 0.
  • traits and pops can be combined in traitsandpops.
