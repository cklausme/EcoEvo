FindEcoEq

FindEcoEq[pops]
finds an ecological equilibrium using initial guess pops.

FindEcoEq[traits, pops]
uses trait values traits and initial guess pops.

DetailsDetails

  • FindEcoEq uses FindRoot to find an ecological equilibrium.
  • The follow options can be given:
  • FindRootOpts{}options to pass to FindRoot
    Fixed{}fixed variables
    BoundaryDetectionFalsewhether to keep within variable Ranges
    ChopTruewhether to apply Chop to the results
    QSSFalsesolve for quasi-steady state (don't fix unlisted vars as zero)
    Timetfix time if nonautonomous model
    VerboseFalseprint out extra info
  • Variables not included in the initial guess are fixed at zero.
  • traits and pops can be combined.
