MatrixToGuildComponents

MatrixToGuildComponents[a, var]
constructs a structured guild based on square matrix a and variable var.

MatrixToGuildComponents[a, var, ncomps]
sets the number of components to ncomps (in case of undefined matrix a).

DetailsDetails

  • MatrixToGuildComponents is useful for setting up matrix population models in SetModel.
  • If ncomps is not given, it is taken from the dimension of a.
  • Variables are incremented from var[1] to var[ncomps].
  • Use # as a placeholder for species number in subscripts.
