EcoEigenvalues

EcoEigenvalues[pops]
returns the the eigenvalues of Jacobian matrix of the ecological equations evaluated at pops.

EcoEigenvalues[traits, pops]
uses trait values traits.

DetailsDetails

  • The following option can be given:
  • NDSolveOpts{}options to pass to NDSolve
    Fixed{}fixed variables
    MultipliersFalsereport Floquet multipliers instead of exponents for ContinuousTime periodic models
    ChopTrueapply Chop to the result
    Timetfix time if nonautonomous model
    VerboseFalseprint out extra info
    VerboseAllFalseapply Verbose recursively
  • traits and pops can be combined in traitsandpops.
