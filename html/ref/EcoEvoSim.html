EcoEvoSim

EcoEvoSim[traits, pops, tmax]
simulates coupled ecological and evolutionary dynamics, with initial traits and pops.

EcoEvoSim[traits, pops, varcovars, tmax]
uses trait variances/covariances in varcovars.

DetailsDetails

  • The following options can be given:
  • Fixed{}fixed variables
    BoundaryDetectionFalseforce traits in traitrange
    NDSolveOpts{MaxSteps -> Infinity}options to pass to NDSolve
    WhenEventOpts{}options to pass to WhenEvent
    DInvOpts{}options to pass to DInv / NDInv
    DelayDInvFalseevaluate NDInv numerically at each point
    EvoEquation"QG"equation for trait dynamics
    FreezeTraitsFalsetreat traits as constants, but accumulate selection
    TraitShiftRate{}change in traits due to environmental change
    TMin0initial time
    OutputTMin0initial time to output
    Output"Dynamics"output type
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for the EvoEquation option are:
  • "QG"quantitative genetics (breeders equation)
    "CE"canonical equation of adaptive dynamics
  • Possible settings for the Output option are:
  • "Dynamics"temporal rule lists
    "FinalSlice"final values
  • Trait variances are given as a list of {V[trait]value} and trait variance/covariance matrices as {G[guild]matrix}. Default is variance of 1 and no covariances.
  • traits and pops can be combined in traitsandpops.
