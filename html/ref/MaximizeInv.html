MaximizeInv

MaximizeInv[traits, sol]
finds the trait value(s) that maximize invasion fitness into a resident community with traits and sol.

DetailsDetails

  • The following options can be given:
  • GuildAutomaticguild invader
    DelayInvFalseevaluate NDInv numerically at each point
    InvOpts{}options to pass to Inv
    MaximizeOpts{}options to pass to Maximize
    ConstraintsAutomaticconstraints on trait values
    Method"NMaximize"method to use
    VerboseFalseprint out extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for Method option are:
  • "Maximize"use Maximize
    "NMaximize"use NMaximize
  • MaximizeInv adds up the number of calls to Inv in $invcount.
  • traits and pops can be combined in traitsandpops.
