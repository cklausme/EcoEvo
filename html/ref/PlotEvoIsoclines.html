PlotEvoIsoclines

PlotEvoIsoclines[{trait1, trait1min, trait1max}]
plots two-species evolutionary isoclines.

PlotEvoIsoclines[sol, {trait1, trait1min, trait1max}]
uses two-species ecological equilibrium sol for improved speed.

PlotEvoIsoclines[{trait1, trait1min, trait1max}, {trait2, trait2min, trait2max}]
allows for two different traits.

PlotEvoIsoclines[sol, {trait1, trait1min, trait1max}, {trait2, trait2min, trait2max}]
uses ecological equilibrium sol.

PlotEvoIsoclines[sol, varcovars, {trait1, trait1min, trait1max}, {trait2, trait2min, trait2max}]
uses trait variances/covariances in varcovars.

DetailsDetails

  • The following options can be given:
  • Fixed{}fixed variables
    Timetfix time if nonautonomous model
    FindEcoAttractorOpts{}options to pass to FindEcoAttractor
    DInvOpts{}options to pass to DInv / NDInv
    PlotOpts{}options to pass to ContourPlot / ListLinePlot
    FrameLabelAutomatichow to label axes
    FitnessGradient"DInv"how to calculate fitness gradients
    ICs{}initial conditions to pass to FindEcoAttractor
    EvoEquation"QG"equation for trait dynamics
    EcoAttractorNumber1which EcoAttractor to use in case of multiple
    ExcludeDiagonalTrueuse Exclusions to exclude the main diagonal
    ESTestTruetest for evolutionary stability
    DelayDInv2Falsere-evaluate DInv numerically at each point for ESTest
    DInv2Opts{}options to pass to DInv / NDInv for ESTest
    IsoclineStyleAutomatichow to style isoclines
    MonitorFalsepopup progress monitor window
    PrintTraceFalseprint variables at each step
    VerboseFalseprint extra info
    VerboseAllFalseapply Verbose recursively
  • With IsoclineStyleAutomatic, evolutionarily stable portions of isoclines are thick, non-evolutionarily stable portions are thin.
  • IsoclineStyle is either {style1,style2} for ESTestFalse or {{ESstyle1,NonESstyle1},{ESstyle2,NonESstyle2}].
  • Possible settings for the FitnessGradient option are:
  • "DInv"analytical DInv
    "NDInv"numerical NDInv
  • Possible settings for the EvoEquation option are:
  • "QG"quantitative genetics (breeders equation)
    "CE"canonical equation of adaptive dynamics
  • When sol is omitted (equivalently, set to "FindEcoAttractor"), FitnessGradient is automatically set to "NDInv" and DelayDInv2 is set to True.
  • Trait variances are given as a list of {V[trait]value} and trait variance/covariance matrices as {G[guild]matrix}. Default is variance of 1 and no covariances.
