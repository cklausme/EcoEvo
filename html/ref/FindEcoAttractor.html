FindEcoAttractor

FindEcoAttractor[]
tries to find an ecological attractor.

FindEcoAttractor[variables]
uses initial guess variables.

FindEcoAttractor[traits]
uses trait values traits.

FindEcoAttractor[traits, variables]
uses traits and variables.

DetailsDetails

  • FindEcoAttractor uses a heuristic algorithm to try to find a single ecological attractor.
  • The following options can be given:
  • MethodAutomaticmethod to find attractor
    ChopTrueapply Chop to final result
    NumTries1number of times to try FindEcoEq with different initial guesses
    SolveEcoEqOpts{}options to pass to SolveEcoEq
    NSolveEcoEqOpts{}options to pass to NSolveEcoEq
    FindEcoEqOpts{}options to pass to FindEcoEq
    EcoSimOpts{}options to pass to EcoSim
    FindEcoCycleOpts{}options to pass to FindEcoCycle
    WarmUp0how long to run EcoSim to refine initial guess (method "FindEcoEq" & "FindEcoCycle")
    TMaxAutomatichow long to run EcoSim (method "EcoSim")
    EqThreshold1/100000threshold for change rates to identify equilibria (method "EcoSim")
    Timetfixed time if nonautonomous model
    PeriodAutomaticinitial guess to period
    MaxPeriod16max period to pass to FindPeriod (method "EcoSim")
    MaxPeriodMultiplier2max multiplier of base period Period
    TestValidityTruewhether to run SelectValid on potential attractors
    TestStabilityTruewhether to run SelectEcoStable on potential attractors
    VerboseFalseprint out extra info
    VerboseAllFalseapply Verbose recursively
  • Possible settings for the Method option are:
  • Automatic"FindEcoCycle" if periodic model and no Time given, "NSolveEcoEq" if no initial guess given or number of equations <=4, "FindEcoEq" if initial conditions given
